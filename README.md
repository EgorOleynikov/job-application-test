<h1>Тестовое задание</h1>

Тестовое задание на должность Junior Python Developer, а так же краткий перечень условий выполнения. Приведенные перечни являются обязательными, введение дополнительных механик взаимодействия с БД одобряются и остаются на Ваше усмотрение. Файл БД прилагается.

Условия:

Должен использоваться язык программирования Python версии 3.6-3.11, библиотека для разработки графических приложений PyQt5 v.5.15 ≥, операционная система Linux (версии ОС: Astra Linux ≥ 1.6, Ubuntu ≥ 20.04, Kubuntu ≥ 20.04).


Задание:

1. Дана БД SQLite.

2. Построить из БД иерархический список (дерево через QTreeView) с возможностью редактирования, добавления дочерних элементов.

2.1. Столбцы id, id_parent, state должны быть скрыты.

2.2 Если в столбце state значение 0 поле name должно иметь красный фон, если 1 - желтый, если 2 - зеленый.

2.3 В поле image находится картинка в формате png.
